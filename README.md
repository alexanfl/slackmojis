# Slackmojis
Lazy way to download every emoji on 
https://slackmojis.com/categories/19-random-emojis/

## Setup
Install `node` and `npm`.

OSX: 
```bash
brew install node
```
installs both

```bash
npm install request-promise 
```

## Usage
To download all emojis on the url to `emojis`
```bash
node app.js
```
